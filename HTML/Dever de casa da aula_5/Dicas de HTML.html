  • Exibição de imagens

Exibir imagens em uma página HTML pode servir a diversos propósitos, incluindo:

  1. Ilustração visual: As imagens podem complementar o conteúdo textual, tornando a página mais atraente e fácil de entender para os visitantes.
  2. Apresentação de produtos: Se você estiver criando uma loja online ou um catálogo, exibir imagens dos produtos permite que os clientes vejam como são os itens que estão considerando comprar.

Código utilizado:

    < img src="caminho/para/imagem.jpg" alt="Descrição da Imagem" >


  • Exibição de uma tabela simples

Exibir tabelas simples em uma página HTML pode ser útil em várias situações:

  1. Apresentação de dados quantitativos: Se você tem dados numéricos que deseja exibir de forma clara e ordenada, uma tabela é uma escolha adequada para apresentar essas informações de maneira estruturada.
  2. Comparação de informações: Quando você precisa comparar diferentes conjuntos de dados ou informações, uma tabela pode facilitar a visualização das diferenças e semelhanças entre eles.

Código utilizado:

        < !DOCTYPE html >
        < html lang="pt-br" >
< head >
    < meta charset="UTF-8" >
    < meta name="viewport" content="width=device-width, initial-scale=1.0" >
    < title>Exemplo de Tabela
< /head >
< body >
    < table border="1" >
        < tr >
            < th >Nome< /th >
            < th >Idade< /th>
        < /tr >
        < tr >
            < td >João< /td >
            < td >30< /td >
        < /tr >
        < tr >
            < td >Maria< /td >
            < td >25< /td >
        < /tr >
    < /table >
< /body >
< /html >

  • Exibição de um link

Exibir links em uma página HTML serve a diversos propósitos importantes:

  1. Navegação: Links são fundamentais para permitir que os usuários naveguem entre diferentes páginas da web, seja dentro do mesmo site ou para outros sites na internet.
  2. Interatividade: Links podem tornar uma página da web mais interativa, permitindo que os usuários cliquem em elementos específicos para obter mais informações ou realizar ações, como enviar um formulário ou iniciar uma transação.

Código utilizado:

        < p >Visite o < a href="colocar o link desejado">Exemplo< /a > para mais informações.< /p >
